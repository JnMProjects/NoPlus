## Button Variants

The `Button` component supports the following variants:

- `default`: The default button variant.
- `destructive`: A button variant for destructive actions.
- `outline`: A button variant with an outline.
- `secondary`: A secondary button variant.
- `ghost`: A button variant with a transparent background.
- `link`: A button variant that looks like a link.
- `expandIcon`: A button variant with an expand icon.
- `ringHover`: A button variant with a hover effect.
- `shine`: A button variant with a shine animation.
- `gooeyRight`: A button variant with a gooey effect on the right side.
- `gooeyLeft`: A button variant with a gooey effect on the left side.
- `linkHover1`: A button variant with a hover effect on the bottom border.
- `linkHover2`: A button variant with a hover effect on the bottom border.

## Button Sizes

The `Button` component supports the following sizes:

- `default`: The default button size.
- `sm`: A small button size.
- `lg`: A large button size.
- `icon`: A button size for icon-only buttons.
