# Item
### The Item Component is used as A Holder for other Components inside itself

## Default
<Canvas of={ItemStories.Default} />

### The Item Component also can be used as A Error Holder

## Errored
<Canvas of={ItemStories.Error} />


## Usage

```tsx
import {
    Item,
    ItemTitle,
    ItemDescription,
} from "@components/layout/"

<Item {/*Optionaly errored*/}>
    <ItemTitle>Just an Example Text</ItemTitle>
    <ItemDescription>
        This is the description
    </ItemDescription>
</Item>
```
